# URL Shortner

A simple and efficient **URL shortening service**. This project allows users to convert long URLs into short, manageable links and redirect them back to the original destination. Built with a clean separation between **Frontend** and **Backend**, it is easy to extend and deploy.

---

## 📂 Project Structure
```
urlShortner/
│── Backend/        # Server-side logic (API for shortening & redirecting)
│── Frontend/       # Client-side interface (HTML, CSS, JS)
│── README.md       # Project documentation
```

- **Backend**: Handles URL shortening, storage, and redirection logic.  
- **Frontend**: Provides a simple user interface to input long URLs and retrieve shortened links.  
- **Languages Used**: **JavaScript**, **HTML**, **CSS**

---

## 🚀 Features
- Shorten long URLs into unique short codes.
- Redirect shortened URLs to their original destination.
- Simple, responsive frontend interface.
- Modular design (Frontend + Backend separation).
- Easy to extend with a database or cloud hosting.

---

## 🖥️ Usage
1. Enter a long URL in the frontend input field.  
2. Submit to generate a shortened link.  
3. Use the shortened link to redirect back to the original URL.  

---

## 🔧 Technologies Used
- **Node.js / Express** (Backend API)
- **JavaScript, React, CSS** (Frontend UI)
- **Database** (MongoDB)
